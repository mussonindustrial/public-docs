Calculates specific isochoric heat capacity.

## Syntax

```python
system.thermo.if97.isochoricHeatCapacity(h, s) # Enthalpy and Entropy
system.thermo.if97.isochoricHeatCapacity(p, h) # Pressure and Enthalpy
system.thermo.if97.isochoricHeatCapacity(p, s) # Pressure and Entropy
system.thermo.if97.isochoricHeatCapacity(p, t) # Pressure and Temperature
```

:::info
This function must be called with named parameters.
:::

### Parameters

<APITable>

| Type     | Parameter | Description                     |
| -------- | --------- | ------------------------------- |
| _Double_ | h         | Specific Enthalpy (J/kg)        |
| _Double_ | p         | Absolute Pressure (Pa)          |
| _Double_ | s         | Specific Entropy (J/(kg·K))     |
| _Double_ | t         | Temperature (K)                |

</APITable>

### Returns
__Double__ - Specific Isochoric Heat Capacity (J/(kg·K))

The heat required to raise the temperature of a unit mass of a substance by one degree at constant volume.

### Scope
Gateway, Designer, Client, Perspective Session

## Example

### Pressure and Temperature
```python
p = 101325
t = 373.15
result = system.thermo.if97.isochoricHeatCapacity(p=p, t=t)
print result
```

1553.54678163
