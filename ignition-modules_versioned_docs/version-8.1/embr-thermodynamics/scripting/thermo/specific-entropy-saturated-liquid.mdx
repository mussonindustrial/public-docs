Calculates the specific entropy for saturated liquid.

## Syntax

```python
system.thermo.if97.specificEntropySaturatedLiquid(p) # Pressure
system.thermo.if97.specificEntropySaturatedLiquid(t) # Temperature
```

:::info
This function must be called with named parameters.
:::

### Parameters

<APITable>

| Type     | Parameter | Description                     |
| -------- | --------- | ------------------------------- |
| _Double_ | p         | Absolute Pressure (Pa)          |
| _Double_ | t         | Temperature (K)                |

</APITable>

### Returns
__Double__ - Specific Entropy for Saturated Liquid (J/(kgÂ·K))

The specific entropy of a liquid in equilibrium at the given pressure or temperature.

### Scope
Gateway, Designer, Client, Perspective Session
